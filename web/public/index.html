Cómo funciona

NewsRadar Verifica es un sistema automatizado de análisis de cobertura mediática. No decide si una noticia es verdadera o falsa: cruza fuentes y detecta patrones.

Proceso

  • Cada hora se recogen ~250 artículos de 13 fuentes RSS internacionales
  • Se analizan 13.000+ registros de GDELT (monitor global de medios)
  • Los titulares se agrupan por similitud semántica para detectar trending topics
  • Cada tema se cruza con Google Fact Check API para ver si ya fue verificado
  • Se aplican señales de riesgo mecánicas: diversidad de fuentes, contradicciones, cobertura
  • Groq (Llama 3.1) analiza diferencias de framing entre fuentes sin decidir verdad

Señales de riesgo

  • SINDICACIÓN MASIVA — Múltiples fuentes con titular casi idéntico
  • SESGO DE FUENTE — Toda la cobertura viene del mismo tipo de medio
  • TENDENCIA DIGITAL — Fuerte en GDELT pero sin cobertura en medios tradicionales
  • REINCIDENTE — Mismo patrón que un tema ya desmentido antes
  • COBERTURA MIXTA — Diferentes líneas editoriales lo cubren de forma distinta

Veredictos

  • VERIFICADO — Google Fact Check confirmó la información
  • FALSO — Google Fact Check encontró un desmentido
  • ENGA&NtilDE;OSO — Contiene información engañosa según verificadores
  • SIN VERIFICAR — No hay verificación externa disponible
